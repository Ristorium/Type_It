# Type_It
Made with Claude AI (Sonnet 4.6)

**English**
How to Use
Make and xlsx file for sentences you want to memorize.
  Column A for sentences you want to memorize.
  Column B for descriptions, translated ones, anything else you'd like to put in there.
Use 힌트 toggle for Hints

**Korean**
사용법
xlsx 파일을 활용해 외우고 싶은 문장들을 적으세요.
  A열에는 외우고 싶은 문장을 적으세요.
  B열에는 A열에 적은 것에 대응되는 설명, 번역된 문장, 혹은 그 어떤 것이든 넣으세요.
힌트 토글을 사용해 힌트를 볼 수 있습니다.
